MODE: Plan
BUILD LOCATION: Viva

Task:
Plan the first lightweight Pending Outreach admin page
inside Viva.

Goal:
Create a simple internal page where admins can view and
manage Marketplace pending outreach records without
needing to rely entirely on pasted UUIDs from the
extension.

Important:
This is a planning phase only.
Do NOT build anything yet.
Do NOT redesign the CRM.
Do NOT build a complex queue system yet.
Do NOT add Messenger scraping.
Do NOT add automatic conversion.
Do NOT add bulk actions.
Keep the first version simple and safe.

Core purpose of the page:
Allow admins to:
- see pending outreach records
- filter by status
- view key Marketplace context
- manually convert records
- manually skip records
- quickly see which records need attention

- /admin/marketplace
